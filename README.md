# To-Do List Web App

A modern, responsive To-Do List application built with HTML, CSS, Bootstrap 5, and Vanilla JavaScript.

## Features

- Add new tasks with title, description, due date, and priority
- Edit and delete existing tasks
- Mark tasks as complete/incomplete
- Filter tasks: All, Active, Completed
- Sort by: Due Date, Priority, Created Time
- Search tasks by keyword
- Light/Dark mode toggle
- Data persistence using localStorage
- Responsive design for all devices
- Optional task export/import as JSON

## Tech Stack

- HTML5
- CSS3
- Bootstrap 5
- Vanilla JavaScript
- LocalStorage for data persistence
- Bootstrap Icons

## How to Run Locally

Simply open the `index.html` file in your web browser. No build process or server required!

## Data Persistence

All tasks are automatically saved to your browser's localStorage. They will persist between page refreshes but are limited to your browser and device.
